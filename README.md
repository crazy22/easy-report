# easy-report
依赖于阿里的easy-excel,开发的配置化报表系统

![image](https://github.com/crazy22/easy-report/blob/master/image/easy-report.png)

### 一、已实现功能

- 根据数据库报表配置信息下载报表
- 上传报表模板，写入数据库
- 修改报表配置信息
- 删除报表配置信息
